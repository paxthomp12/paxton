# Paxton Thompson Portfolio

Professional portfolio website for Paxton Thompson - Baseball Pitcher, Wealth Management Intern, and Co-Founder of Old Logan Capital.

## Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/paxthomp12/paxton.git
cd paxton
```

2. Install dependencies:
```bash
npm install
```

### Running the Development Server

Start the live-server development server:
```bash
npm start
```

This will:
- Start a local server at http://127.0.0.1:8080
- Automatically open the site in your default browser
- Watch for file changes and auto-reload the page

### Project Structure

```
paxton/
├── index.html          # Main portfolio page
├── assets/
│   └── images/        # Profile photos and media
├── package.json       # Node.js dependencies and scripts
├── .gitignore        # Git ignore rules
└── README.md         # This file
```

### Development

The site is a single-page HTML portfolio with embedded CSS and JavaScript. To make changes:

1. Edit `index.html` directly
2. Save your changes
3. The browser will automatically reload

### Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- live-server for development
